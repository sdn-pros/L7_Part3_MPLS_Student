#### Ethernet Interfaces Device Configuration

```eos
!
interface Ethernet1
   description P2P_LINK_TO_BL1-DC2_Ethernet1
   no shutdown
   mtu 1500
   no switchport
   ip address 10.0.0.83/31
   mpls ip
   isis enable CORE
   isis circuit-type level-2
   isis metric 50
   no isis hello padding
   isis network point-to-point
!
```
